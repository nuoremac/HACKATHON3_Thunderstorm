# Campus Radar

Next.js frontend prototype for the Hackverse 24H challenge.

## Stack

- Next.js 15.5.15+
- React 19
- TypeScript
- Tailwind CSS
- shadcn-style UI primitives
- Recharts
- Mocked campus data

## Run Locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## Product Scope

The frontend demonstrates:

- student radar dashboard;
- recommendation scoring and confidence;
- Twist 01 assumption ledger;
- new-student exploration mode;
- impact profile;
- contextual requests instead of generic chat;
- event and association ranking;
- admin analytics and 90-day adoption plan.
